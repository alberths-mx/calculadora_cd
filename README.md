## Calculadora de Ciencias de Datos

## Este proyecto es una calculadora desarrollada 100% en Python + Streamlit, diseñada como práctica para estudiantes de Ciencia de Datos. La aplicación permite realizar operaciones matemáticas, estadísticas, gráficas, procesamiento de imágenes en un entorno sencillo y visual.

Pasos: 

## En la terminal de cursor ingresa a la ruta donde se va a realizar a alojar el proyecto.
ej. cd C:\Users\mia_d\OneDrive\LOG\802\Albert\
## Crea la(s) carpeta(s) / subcarpeta(s) donde será lojado el repositorio
ej. mkdir Albert
cd Albert
mkdir projects
cd projects
mkdir calculador
cd calculator
mkdir DataScienceCalculator
cd DataScienceCalculator

## Clona el repositorio
git clone https://github.com/UNRCProjects/DataScienceCalculator.git

# Crear el entorno virtual de Python
python -m venv venv

# Ejecuta un script de Windows para activar el entorno virtual de Python
venv\Scripts\activate

# Instala las instancias necesarias (streamlit, sympy y numpy).
pip install -r requirements.txt

# ejecuta la app dentro de una página web
streamlit run app.py
